A choose-your-own-adventure game designed to teach players about fire safety through a variety of fun and engaging scenarios, all themed around moving into student residence. The goal of this project was to make an engaging and informative game that students would want to play, which makes it a valuable tool for retaining this life-saving information.

I created the game itself using the Unity 2D game engine, and all of the programming was done using C# in Visual Studio. I made custom scripts for all of the game's logic, such as displaying pages, handling branching paths and defining button behavior. The game is published as a windows app and a web app, which is available on my personal Itch.io page at www.constantine.games

I was not involved with the process of procuring the images used for this project, but it is my understanding that we used various creative commons images for some of the scenarios. 

This is the project for this course that I'm most proud of. Not only do I think it represents a large amount of personal effort and dedication, it is a polished and well-designed finished project that serves a real benefit to those who play it. I also think that the collaborative process was positive and productive. Creating the codebase for this game also helped me write a whole system that could be used to create similar educational games in the future.

A great deal of effort and detail went into the creation of this game in regards to accessibility. All the colours went passed a colorblind checker. A colorblind mode was written so that colours could easily be changed to make versions for all manner of different visual impairments. Standard text using Unity's new official TextMeshPro text objects ensure that screen reader implementation should be straightforward. Adjustments were made at every step of the process. The game was playtested to ensure that displays properly on displays of any size or even some mobile displays. Adjustments were made to text spacing, kerning, tracking, and alternate text colours. Unity New Input System usage means that this game could be easily ported to use input methods more accessible to those who can't use a touchscreen or mouse and keyboard. Accessibility was on the front of my mind when I made this game.
