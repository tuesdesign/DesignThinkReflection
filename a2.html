A game designed to be used by students to practice multiplication tables. The game quizzes the player on multiplication questions ranging from the 1 table to the 12 table, as is standard in the curriculum. Along the way, the game displays an increasing score and positive messages 

I created this game using the Unity 2D Game Engine. The game's scripts were written in C#, programmed in the Visual Studio IDE. The game was published for web, and you can play it on my website at www.constantine.games.

There are no external images or artwork in this game. 

As this was the first group project of the class, I had a difficult time joining a group. The group I did join didn't contribute, so I made the game on my own. I think, despite those circumstances, that the game is well designed, and would be an effective tool to practice multiplication tables.

The Game was designed to be as accessible as possible. I ran all of the colours, the graphic elements, and the font all through an accessibility checker for visual impairment and colourblindness. I used the Unity New Input System to ensure that various input methods could be added for those with motor impairments or for those who cannot use a mouse and keyboard.
