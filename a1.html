This is a website made for an organization called 'Upcycle My Fit'. This organization aims to reduce clothing sent to landfills by ‘upcycling’ clothing and clothing materials into new repurposed designs. They create pieces that are designed as “fashion-forward”.

This design was created to cater to different types of visitors to the site: People who want to donate clothes to the organization, people interested in learning more about the concept of upcycling, and finally customers for the digital catalogue of the store itself.

The primary use of external sources for this project were a series of royalty-free creative commons images of upcycled clothing. These were used as examples in the website because I don't own any upcycled clothing to photograph. Visual sections of the website were created using Mobirise HTML/CSS blocks (This one is, too). The final website his hosted on my personal website using GitHub hosting.

I think that this assignment was a really great showcase of my individual design skills. I'm rather happy with the end result, and I think that this design could be used by this organization with very little alteration, if they really existed.  

Accessibility is a crucial part of web design. I ran my colour scheme and final design through colorblindness checkers and used a highly readable font. I also ensured that the site displays properly on displays of many common resolutions, and that it formats correctly on mobile phones. For mobile, I also included a "hamburger menu" style interface and touchscreen-optimized buttons.
